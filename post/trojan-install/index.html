Trojan 安装及使用记录

2020年7月2日
| 2020

常用的翻墙工具里, 我用过的有Shadowsocks, V2Ray和Trojan. V2Ray测试过后感觉并不好, 目前以Shadowsocks为主力, Trojan作为辅助. 这里记录一下Trojan的使用情况.

安装

Ubuntu 18.04

sudo bash -c "$(curl -fsSL https://raw.githubusercontent.com/trojan-gfw/trojan-quickstart/master/trojan-quickstart.sh)"

Ubuntu 19.04及以上

sudo apt-get install trojan

配置

证书配置

使用acme来申请配置证书

  • curl https://get.acme.sh | sh
  • mkdir ~/public_html
  • sudo python3 -m http.server 80 &
  • acme.sh --issue -d example.com -w /home/username/public_html

Trojan Config

  • 更新配置文件/usr/local/etc/trojan/config.json里的密码及证书相关的配置
    • 注意: 配置文件里的cert路径需填写full chain certs的路径
  • sudo systemctl enable trojan
  • sudo service trojan restart
