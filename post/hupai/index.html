拍牌

2020年6月30日
| 2020

帮朋友拍沪牌, 做个笔记:)

拍牌规则

上海的拍牌规则是10:30~11:00第一轮出价, 之后11:00~11:30进行第二轮出价. 第一轮出价按照指导价格出就好了, 只是一个报名的作用. 重点在第二轮出价, 第二轮出价可以出2次. 但只有在最后几秒进去, 接近中标价格的才能中.

  • 最佳提交时间分布在53.5~57.5s之间, 太早不会中, 太晚服务器不接受

拍牌策略

  • 30s的时候出第一次价格+300看一下验证码, 50秒的时候出第二次价格+600. 手速要快.
  • 50秒出完价格P后, 看价格, 到达P-300的时候就点出价, 到了57秒还没到P-300, 也得出了, 再不出价格进不去了.

拍牌脚本

57秒强制提交脚本, 这样可以在最后集中精神看价格.

import time
import pyautogui

RELEASE = True
print("Good Luck!")

while True:
    if (RELEASE):
        ## Wait for 11:29:57
        while(time.time() % 60 < 57.2):
            time.sleep(0.1)
    else:
        input("Press enter when it's 11:29:10: ")
        ## Wait for 47 seconds
        time.sleep(47-0.2)

    pyautogui.click()
    print(time.asctime(time.localtime(time.time())))
    time.sleep(0.6)
